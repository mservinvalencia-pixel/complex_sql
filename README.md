# Complex SQL

This repository contains advanced SQL examples designed for analytics and data engineering portfolios.

## Purpose

The goal of this repository is to demonstrate practical SQL skills used in real-world analytical workflows, including:

- multi-CTE query design
- JSON field extraction
- joins across normalized datasets
- timestamp transformation and timezone conversion
- derived business segmentation
- aggregation for reporting purposes

## Example included

### `transaction_reporting_example.sql`

This query demonstrates how to:

- extract values from semi-structured JSON fields
- combine transactional and attribute-based tables
- derive reporting dimensions from identifiers
- classify records into reporting segments
- calculate transaction counts and aggregated amounts

## Notes on confidentiality

All table names, field names, mappings, and business labels in this repository have been anonymized for confidentiality.  
The examples are intended to showcase technical patterns and SQL problem-solving approaches without exposing proprietary business logic or internal data structures.

## Skills demonstrated

- SQL
- Common Table Expressions (CTEs)
- JSON parsing
- Aggregations
- Conditional logic with `CASE`
- Analytical query structuring
- Reporting-oriented transformations
